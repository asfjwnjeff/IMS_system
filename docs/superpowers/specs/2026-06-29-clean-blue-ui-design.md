# Clean Blue UI 设计方案

## 概述

保险管理系统全面 UI 优化，统一为 "Clean Blue" 风格（Ant Design 优化版）。
深色侧边栏 + 浅色主体布局保留，提升交互反馈和视觉清晰度。

## 设计系统

### 色彩
- 侧边栏: `#001529`
- 页面底: `#f5f5f5`
- 卡片: `#ffffff`，阴影 `0 1px 2px rgba(0,0,0,0.04)`
- 品牌蓝: `#1677ff`
- 成功绿: `#52c41a`
- 危险红: `#ff4d4f`
- 文字: `#262626` / `#595959` / `#8c8c8c`
- 边框: `#f0f0f0` / `#d9d9d9`

### 圆角
- 统一 6px（卡片、按钮、输入框、标签）
- 弹窗 8px

### 交互
- 所有过渡 0.2s ease
- 悬浮：按钮加深、表格行变蓝底 `#e6f4ff`、卡片阴影加深
- 状态标签：pill 形状，浅底+彩色边框

## 各页面改动

| 页面 | 改动 |
|------|------|
| 侧边栏 MainLayout | 保留深色，选中态品牌蓝高亮，菜单悬浮反馈 |
| 投保申请表 | 行悬浮、斑马纹、标签 pill 化、间距统一 |
| 投保单详情 | Swiss 网格→Ant Descriptions+Card，4列响应式 |
| 投保单编辑 | Swiss 网格→Ant Form+Card，Row/Col 4列，fields.tsx 共享 |
| 保费汇率配置 | 表格行悬浮、弹窗表单圆角统一 |
| 保费费率配置 | 同上 |
| 报案理赔管理 | 行悬浮、标签 pill、弹窗圆角 |
| 新增报案 | 内嵌 Card 间距统一 |
| 报案详情 | Descriptions 2列+Card 分段 |

## 技术实现

- 全局 CSS 覆盖 Ant Design token
- 详情/编辑页删除 `swiss.css`，改用 Ant 组件
- `fields.tsx` 保留作为字段配置共享
- 不改变任何功能和路由
